# GZCTF-Auto
  
## 项目介绍&使用方法
GZCTF 一键部署  
现已支持单 `docker` 以及 `docker+k3s` 部署  
更多功能持续更新......  
测试环境：ubuntu 22.04  
使用方式：  
```
wget -O install.sh https://cdn.moran233.xyz/https://raw.githubusercontent.com/MoRan23/GZCTF-Auto/main/install.sh && chmod +x install.sh && ./install.sh
```
